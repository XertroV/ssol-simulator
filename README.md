# Open SSOL

`Open SSOL` is a desktop game/simulator inspired by the idea of collecting light-speed-lowering orbs and then heading for the finish.

If you are here to play it, the most important thing is the releases page:

- Download the latest build from [Releases](https://github.com/XertroV/ssol-simulator/releases)
- Pick the archive for your platform:
- `Windows x86_64`
- `Linux x86_64`
- `macOS arm64`

## Quick Start

1. Download the archive for your platform.
2. Extract it fully.
3. Open the extracted folder.
4. Launch the game from inside that folder.

Important: keep the executable next to the bundled `assets/` directory. If you move only the executable somewhere else, the game will not be able to load its data correctly.

## Goal

Collect all visible orbs in the level.

After that, the white finish arch becomes active. Pass through it, or click when prompted, to finish the run.

## Default Controls

- `W A S D`: move
- Mouse: look around
- `Escape`: open or close the pause menu
- `F11`: toggle fullscreen
- `Backspace`: reset the run
- `C`: toggle free camera
- `Space` / `Left Shift`: move free camera up / down
- `F7`: toggle the performance HUD

You can change keybinds and several settings from the in-game pause menu.

## In-Game Settings

The pause menu lets you adjust:

- mouse sensitivity
- free camera speed
- master, music, and SFX volume
- fullscreen
- VSync
- performance HUD
- physics gizmos
- desaturation

## Notes

- Releases are packaged as archives, not single standalone executables.
- The game is meant to be launched from the extracted release folder.
- If something looks wrong after extracting, make sure the `assets/` folder is present beside the executable.
